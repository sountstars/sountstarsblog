## 正则方法
```js
reg.test(str)  //检验 
//字符串中有没有正则能匹配到的内容，返回值: true false
reg.exec(str)  //捕获 
//方法用于检索字符串中的正则表达式的匹配。 
//返回一个数组，其中存放匹配的结果。如果未找到匹配，则返回值为 null。
str.match(reg) // 把匹配到的字符放入到数组中，不匹配返回null
str.replace( reg ,""替换成什么 | callback（替换成什么）)
```
